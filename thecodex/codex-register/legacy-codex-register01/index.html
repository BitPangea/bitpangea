Historical Record

This register preserves the original sequential organization of the first five adopted Codices prior to the adoption of the Domain Architecture. These entries remain historically authentic but have since been reorganized within the Structural (ST) and Relational (RL) Domains.

The Codex Register records formal entries adopted into The Codex. Each entry preserves a foundational concept within BitPangea before later systems are built upon it.

Register Principle

The Codex Register does not define the world. It records the definitions adopted into The Codex.

Established Codex Entries

Each adopted Codex is preserved as a permanent record within the Cartographic Period.
